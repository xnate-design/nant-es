## Description

抽象操作[IsCallable](https://tc39.es/ecma262/#sec-iscallable)接收参数（ECMAScript语言值），并返回包含Boolean值，如果是参数是可调用的函数，则一定
有内部方法`[[Call]]`。

## Signature

```ts
function isCallable(it: LanguageType): Boolean
```


## Examples

<Sandpack>
  ```js
  import { isCallable } from '@nant-es/internal';
  console.log(isCallable(null));
  console.log(isCallable(() => {}));
  console.log(isCallable({}));

  ```
</Sandpack>
